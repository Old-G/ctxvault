# CtxVault

> Your AI agents have amnesia. CtxVault is the cure.

Persistent, intelligent memory for AI coding agents (Claude Code, Cursor, Codex, Windsurf).

Git-native memory layer that stores project knowledge as Markdown files, indexed via SQLite with FTS5, and integrates through three layers: Hooks (deterministic), Skills (intelligent), and MCP (universal).

## Status

Under active development. Not yet published to npm.
